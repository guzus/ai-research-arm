## Issues Found

### 1. Bluesky workflow has been silently broken — 0 posts/day
`research/bluesky/2026-05-12.md` (and every recent day) contains only
`_No posts ingested this run: all five pre-fetched feed files [...] returned a 403 Forbidden HTML page instead of JSON_`.

**Root cause** (verified by curl from this runner): `public.api.bsky.app`
returns `HTTP 403 Forbidden` with an HTML body for unauthenticated calls.
The alternate host `api.bsky.app` (no `public.` prefix) serves the same
xrpc endpoints and returns `200 OK` JSON.

| Endpoint | HTTP | Bytes |
|---|---|---|
| `public.api.bsky.app/xrpc/app.bsky.feed.searchPosts` | 403 | 2,334 (HTML) |
| `api.bsky.app/xrpc/app.bsky.feed.searchPosts` | 200 | 7,444 (JSON) |
| `api.bsky.app/xrpc/app.bsky.feed.getAuthorFeed` | 200 | 4,349 (JSON) |

### 2. Hourly RSS workflow has produced zero output for 16 days
`research/rss/` has files through `2026-04-27.md` and nothing newer. Three
of the eleven configured feed URLs returned non-200 status — and because
`curl -sL` silently writes the response body to disk regardless of HTTP
status, those slots filled with HTML error pages that the LLM step couldn't
parse. Around 2026-04-27 the workflow stopped committing anything at all.

Verified broken URLs (curl from this runner):

| URL | HTTP | Fix |
|---|---|---|
| `http://googleaiblog.blogspot.com/atom.xml` | 301 → feedburner | Replaced with `blog.research.google/atom.xml` |
| `https://ai.meta.com/blog/rss/` | 404 | Replaced with `engineering.fb.com/category/ml-applications/feed/` |
| `https://venturebeat.com/category/ai/feed/` | 308 → `/feed` | Works behind `-L`; kept, but now status-validated |

### 3. RSS coverage narrower than what's available
Several high-signal AI feeds are not subscribed today: DeepMind's own blog
(separate from research.google), Cohere, MIT Technology Review, arXiv cs.CL,
and the major analyst newsletters (Latent Space, Import AI, One Useful
Thing, Stratechery, AI Snake Oil, Simon Willison). The 2026-05-12 digest's
"Sources" footer cites WebSearch and hand-fetched links instead of the RSS
surface — a clear coverage gap.

### 4. Bluesky surface is generic-query–dominated
The current workflow runs 4 keyword searches plus one `from:karpathy`
search. Generic searches return high-volume / low-signal posts; named
author feeds yield much better signal-per-byte for known researchers.

## Changes Made

### `.github/workflows/2h-bluesky.yml`
- Switched base URL from `public.api.bsky.app` → `api.bsky.app`.
- Added `Accept: application/json` and a User-Agent header.
- Added per-fetch JSON-shape validation; non-JSON responses are replaced
  with `{"posts":[]}` so the downstream LLM step always sees uniform data.
- Added `getAuthorFeed` pulls for `karpathy.bsky.social`,
  `simonwillison.net`, and `ylecun.bsky.social`, with the response normalized
  to the same `{"posts":[...]}` shape as search results.
- Removed the broken `from:karpathy` search-based fetch.
- Added an end-of-step summary that prints post counts per file.

### `.github/workflows/hourly-rss.yml`
- Replaced inline `curl -sL ... 2>/dev/null` calls with a `fetch_feed()`
  helper that validates HTTP status (must be 2xx) and feed shape (must
  contain `<rss>` / `<feed>` / `<rdf>`). Broken feeds are now replaced
  with an empty `<rss></rss>` stub and visibly logged with `[!]`, so
  future breakage is observable in workflow logs within one hour.
- Fixed Google AI Blog URL (`googleaiblog.blogspot.com` → `blog.research.google`).
- Fixed Meta AI Blog URL (`ai.meta.com/blog/rss/` → `engineering.fb.com/category/ml-applications/feed/`).
- Added 9 new feeds:
  - **DeepMind** (`deepmind.google/blog/rss.xml`) — separate from Google Research.
  - **Cohere** (`cohere.com/blog/rss.xml`) — frontier lab, currently untracked.
  - **arXiv cs.CL** — NLP submissions; complements cs.AI / cs.LG.
  - **MIT Technology Review** — long-form AI coverage.
  - **Latent Space** (swyx) — practitioner-focused weekly.
  - **Import AI** (Jack Clark) — Anthropic policy lead's weekly.
  - **One Useful Thing** (Ethan Mollick) — applied-LLM analysis.
  - **Stratechery** (Ben Thompson) — strategy / industry context.
  - **AI Snake Oil** (Narayanan / Kapoor) — academic critique.
  - **Simon Willison's blog** — daily LLM commentary + tools.
- Updated the Claude prompt to enumerate the new files and added output
  sections for Cohere, MIT Tech Review, the analyst-newsletter section,
  and arXiv cs.CL.
- End-of-fetch summary prints item count + byte size per feed.
