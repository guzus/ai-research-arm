**xAI** (now operating as the **SpaceXAI** division inside SpaceX since the February merger) launched its **first terminal-based agentic coding tool, Grok Build**:

- **What it is:** **Terminal-native CLI** for "professional software engineering and complex coding work." Generates code, runs tests, fetches docs, executes shell commands from natural-language prompts.
- **Model:** **Grok 4.3 beta** underlying. **~1M-token context window** per spec sheet (some outlets cite up to **2M** under the 16-agent Heavy architecture, allowing a small-to-medium codebase entirely in-context).
- **Architecture:** **Plan / Search / Build** three-stage workflow. Up to **8 parallel sub-agents** (16 in Heavy mode). Plan-mode with **step-by-step approval**, **diffs before every change**, **headless mode for scripts**, and **user confirmation for irreversible commands**.
- **Compatibility:** Designed to be **Skills- and MCP-compatible with Claude Code** — lets developers reuse Anthropic-ecosystem tooling and skills with Grok Build. xAI is explicitly **lowering switching costs** rather than building a closed format.
- **Pricing:** Initially **SuperGrok Heavy at $300/month** at `build.grok.com`. A new **SuperHeavy tier** opens with a **$99/month introductory price for six months** (then $299/month).
- **Limitations:** **Grok 4.3 has not undergone independent benchmarking yet**, and xAI flags **expected bugs in early beta**.

**Read-through:** Three implications. (1) Grok Build is **xAI's first direct salvo at Anthropic's Claude Code monetization moat** — and the **Skills/MCP compatibility** posture is a meaningful concession: xAI is acknowledging Claude Code's spec layer as the de-facto standard for agentic-coding interop, mirroring what OpenAI did with the AGENTS.md format. (2) The **$99-introductory / $299-standard pricing** undercuts Claude Code's **$200 Max plan** on the floor but matches it on the ceiling, while bundling the broader Grok 4.3 stack — putting near-term **pricing pressure on Anthropic Claude Code and OpenAI Codex** in the contested individual-developer segment. (3) Coming days after the **May 14 PM "all xAI co-founders have exited" Information scoop**, the Grok Build launch is xAI's **execution counter-narrative** — but the load-bearing question is whether Grok 4.3 actually matches Claude Opus 4.7 / GPT-5.5 on code benchmarks once independent results land.
