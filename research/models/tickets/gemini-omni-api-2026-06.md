**Gemini Omni Flash** reached **API general availability** around
**2026-06-30/07-01**, bringing Google's text-prompt **video
generation/editing** to the Gemini API for the first time — reported in the
2026-07-01 daily digest (@testingcatalog, Ars Technica, The Decoder)
alongside the Nano Banana 2 Lite image launch
([[google-nano-banana-2-lite-2026-06]]).

**Why a new ticket, not a reopen.** Gemini Omni ([[gemini-omni]]) launched
at Google I/O 2026 (2026-05-19) in the Gemini app, Flow, and YouTube with
API support "coming soon," and was **closed released-and-aged** on
2026-06-16. That ticket's transition triggers explicitly said to **reopen
via a new ticket if the Gemini API GA lands** — which it now has. This
ticket carries the API-availability lifecycle; the original I/O launch stays
on the closed ticket.

**Confirmed vs. thin.** The event is reported by trade press, so `released`,
but **`verification: partial`**: coverage is secondary and thin this cycle —
no official Google/DeepMind post text, pricing, quota, or model id was
surfaced. Those upgrade verification toward confirmed when they land.

**Transition triggers:**
- Official Gemini API post / pricing / model id → UPDATE, advance verification.
- ≥4 weeks settled into normal coverage → close `released-and-aged`.

**Dedup note:** further Gemini Omni API pricing, quota, or capability signal
UPDATES this ticket; the image-native Nano Banana line stays on its own
tickets ([[google-nano-banana-2-lite-2026-06]]).
