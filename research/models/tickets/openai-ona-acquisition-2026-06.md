**OpenAI announced on 2026-06-11 that it will acquire Ona**, a deal aimed at
giving **Codex** secure, **persistent cloud environments** for long-running
enterprise agents. Where Codex today shines at interactive, session-bound
coding, Ona's contribution is **durable, stateful infrastructure** —
environments that persist across sessions so an agent can carry a multi-hour
or multi-day task without losing its working context.

**Why it's tracked here.** This is a discrete **acquisition** that materially
shapes OpenAI's agent roadmap — the event class this timeline tracks alongside
