NVIDIA introduced Cosmos 3 Edge, an open frontier world model built to run
on-device for robotics, autonomous vehicles, and live video agents. NVIDIA
published the full weight set, post-training recipes, and code on Hugging
Face at launch rather than gating it behind an API, continuing the Cosmos
line's open-release pattern.

The model is pitched at roughly 4B parameters, aimed at letting robots and
edge devices "learn and act" locally without a cloud round-trip. Multiple
independent accounts (including NVIDIA's own official amplification)
corroborated the release the same day.
