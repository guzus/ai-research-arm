Microsoft and AMD announced new Azure AI/HPC infrastructure built on
AMD's "Helios" rack-scale system — MI455X GPUs, EPYC "Venice" CPUs,
Pensando DPUs, and ROCm software — with new VM families (HDv2, HXv2, ND
MI455X v7) reportedly shipping in H2 2026. The official @Azure account
posted about it directly, and multiple financial-news accounts
independently corroborated the same details the same day.
