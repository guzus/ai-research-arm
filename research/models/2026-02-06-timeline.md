## Intelligence Notes

### Day-After Analysis: Dual Frontier Launch Impact (Feb 6)

- **Claude Opus 4.6 and GPT-5.3-Codex released within 30 minutes of each other on Feb 5.** This remains the most significant same-day frontier model release in AI history. Both companies dropped major upgrades simultaneously. Initial benchmark comparison wars continue on social media.

- **Platform availability widening**: Opus 4.6 now confirmed available on Amazon Bedrock, Azure Foundry, and GitHub Copilot (GA). GPT-5.3-Codex available via ChatGPT, Codex app, CLI, and IDE extensions with API access "coming soon."

- **Benchmark overlap remains minimal** — only Terminal-Bench 2.0 and OSWorld appear in both companies' reported metrics. GPT-5.3-Codex leads Terminal-Bench 2.0 (77.3% vs 65.4%); Opus 4.6 leads OSWorld (72.7% vs 64.7%) and SWE-bench Verified (80.8% vs 56.8% SWE-bench Pro — different variants). Industry consensus: **Codex is the better terminal agent; Claude is the better computer use agent and generalist.**

- **Cybersecurity concern**: Fortune reports GPT-5.3-Codex is the first OpenAI model with "unprecedented cybersecurity risks" — the first model to receive a "high" cybersecurity rating on OpenAI's preparedness framework, meaning it could enable real-world cyber harm if automated at scale. OpenAI committed $10M in cyber defense API credits.
