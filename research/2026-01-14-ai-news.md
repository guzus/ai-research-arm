### DeepSeek Publishes Revolutionary AI Training Papers (January 13, 2026)
DeepSeek, the Chinese AI startup based in Hangzhou, published two groundbreaking technical papers on **January 13, 2026**, introducing innovative training methods to overcome GPU memory limitations amid US chip export restrictions:

- **Engram Technique**: Introduces "conditional memory" that separates basic facts from complex computations, reducing reliance on GPU high-bandwidth memory (HBM) and enabling larger models without instability
- **Manifold-Constrained Hyper-Connections (mHC)**: Allows richer internal model communication while preserving training stability and efficiency as models scale

Industry analysts are calling mHC a "striking breakthrough" for bypassing compute bottlenecks. These techniques signal progress toward DeepSeek's next flagship models, including the anticipated **DeepSeek-V4** with revolutionary coding capabilities expected around mid-February 2026.

## Security & Infrastructure

### Microsoft Patches Critical Copilot "Reprompt" Exploit (January 13, 2026)
Microsoft patched a critical **"Reprompt"** security vulnerability in Copilot Personal allowing attackers to hijack sessions and exfiltrate user data via phishing links and URL parameter injection.

**Vulnerability Details:**
- **Attack Vector:** Parameter-to-Prompt (P2P) injection via "q" URL parameter
- **Capabilities:** One-click data theft (recent files, user location) without user input beyond clicking phishing link
- **Scope:** Affected Copilot Personal (consumer version); Microsoft 365 Copilot unaffected
- **Timeline:** Reported August 31, 2025; fixed January 13, 2026 Patch Tuesday
- **Status:** No wild exploitation detected

**Action Required:** Users should apply latest Windows updates immediately
