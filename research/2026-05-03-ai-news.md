### Microsoft **Agent 365** Hits General Availability — AI-Agent Control Plane Lands With Cross-Cloud Discovery (AWS Bedrock + Google Cloud) and OpenClaw Local-Agent Support
**Date:** Friday, May 1, 2026 | **Sources:** [Microsoft Security Blog — Microsoft Agent 365, now generally available, expands capabilities and integrations](https://www.microsoft.com/en-us/security/blog/2026/05/01/microsoft-agent-365-now-generally-available-expands-capabilities-and-integrations/), [Thurrott — Microsoft Agent 365 Platform Goes Out of Preview and Adds Support for Local AI Agents](https://www.thurrott.com/a-i/335594/microsoft-agent-365-platform-goes-out-of-preview-and-adds-support-for-local-ai-agents), [Winbuzzer — Microsoft Agent 365 Hits General Availability With Local AI Agent Controls](https://winbuzzer.com/2026/05/02/microsoft-agent-365-general-availability-local-ai-agents-xcxwbn/), [Let's Data Science — Microsoft Launches Agent 365 Out of Preview](https://letsdatascience.com/news/microsoft-launches-agent-365-out-of-preview-f33e1165), [M365 Admin — Microsoft Agent 365 Generally Available May 1, 2026](https://m365admin.handsontek.net/microsoft-agent-365-generally-available-may-1-2026/), [Nerd Level Tech — Microsoft Agent 365 Goes GA: AI Agent Control Plane 2026](https://nerdleveltech.com/microsoft-agent-365-ga-ai-agent-control-plane), [Microsoft Learn — Microsoft Agent 365 overview](https://learn.microsoft.com/en-us/microsoft-agent-365/overview), [PRNewswire — Zensai Introduces Human Success Agent for Microsoft Agent 365](https://www.prnewswire.com/news-releases/zensai-introduces-human-success-agent-for-microsoft-agent-365-redefining-ai-driven-learning-and-human-success-in-the-flow-of-work-302760146.html)

Microsoft moved **Agent 365**, its enterprise control plane for AI agents, from public preview to **general availability on Friday, May 1, 2026** — first announced at Ignite 2025 in November and widely viewed as Microsoft's structural answer to the **agent-sprawl governance problem** that has dominated CIO conversations since OpenAI's ChatGPT Agent and Anthropic's Computer Use both shipped Q4 2025. The May 2 report did not capture this launch.

- **Pricing:** **$15/user/month** standalone, or bundled in **Microsoft 365 E7** SKU (the new top-tier E-class license introduced for Agent 365 distribution).
- **Cross-cloud agent discovery (public preview at GA):** **registry sync with AWS Bedrock and Google Cloud agents** — IT teams can now **automatically discover, inventory, and govern** AI agents created on rival clouds from a single Microsoft pane. Lifecycle controls (**start, stop, delete**) follow shortly per the Microsoft Security Blog post.
- **Local-agent support:** GA introduces support for **agents created with OpenClaw** (the open-source agent runtime from Anthropic alumni; covered in earlier reports), letting Agent 365 govern on-device/edge agents alongside cloud-hosted ones.
- **Microsoft Defender integration** scheduled for **June 2026:** Defender for Cloud will provide **asset-context mapping per agent** — devices the agent runs on, MCP servers configured, identities, and reachable cloud resources.
- **Ecosystem launch partners:** **Zensai** (Human Success Agent), **Workday**, **ServiceNow**, **SAP**, **Salesforce**, and **Atlassian** all shipped Agent 365–compliant agents at GA.

**Read-through:** Three implications. (1) **Agent 365's cross-cloud discovery is the single most consequential enterprise-AI-governance feature shipped in 2026 to date** — it operationalizes "**bring-your-own-agent**" at the same compliance posture as Microsoft Entra ID for human identity, and gives Microsoft a structural advantage no rival hyperscaler can match because **AWS and Google must show up inside Agent 365's inventory whether they like it or not**. AWS Bedrock Managed Agents and Google Vertex Agent Engine are now *governed assets*, not competing platforms. (2) **The pricing positions Agent 365 as a near-mandatory compliance line item for any Fortune-1000 CIO managing agent sprawl** — the $15/user/month tier is intentionally below the $30/month Copilot tier, making it a **passive ride-along on every existing M365 license**. Expect **>$2B Agent 365 ARR within 12 months** based on Q1 2026 Copilot adoption curves. (3) **Anthropic and OpenAI must respond with parallel cross-cloud governance products** — both labs ship agents but neither offers a multi-cloud governance plane. Watch for **Anthropic Claude Console multi-cloud governance** in the Code with Claude SF May 6 keynote.

**MoonPay** announced **MoonAgents Card** on **Friday, May 1, 2026** — a **virtual Mastercard debit card** that lets users **and AI agents** spend stablecoins (initially **USDC on Solana**) directly from **self-custodial onchain wallets** at any Mastercard-accepting merchant worldwide.

- **Architecture:** self-custodial wallet ↔ Mastercard virtual card via **Monavate** (regulated Visa/Mastercard/Discover principal member). User authorizes a **smart contract** that releases stablecoin balance at the moment of transaction.
- **AI-agent-first framing:** the product's explicit thesis is that **"agents are already managing wallets, executing trades, and moving value onchain — the one thing they couldn't do was spend at a merchant. Now they can."** This is the **first fiat-rail card explicitly designed to be controlled by an LLM agent** rather than retrofitted from a human-issued card.
- **Launch wallet partner:** **Exodus Movement** (Nasdaq-listed self-custody wallet vendor).
- **Geographic availability:** **MoonPay CLI** in **UK and LATAM** today; **U.S. and EU** in coming months.
- **Strategic context:** lands the same week as the **ClawBank Manfred AI agent autonomous incorporation** (below), **Justin Sun's B.AI crypto-AI gateway** (below), and the broader x402 / ERC-8004 / agent-payments stack maturation.

**Read-through:** Three implications. (1) **MoonAgents Card is the first true "agent-first" payment instrument in production**, and it operationalizes a regulatory-grey-zone scenario the Visa/Mastercard/Apple Pay-Treasury 2025 working group has been litigating since Q3 2025 — **who is the cardholder of record when an LLM authorizes the swipe?** MoonPay's Monavate-routed answer is "the human who funded the wallet, mediated by smart contract" — and that legal architecture will become the **template for U.S. agent-payment compliance** if it survives an enforcement test. (2) **The Mastercard imprimatur is the larger story than the MoonPay product itself** — Mastercard publicly partnering on an AI-agent-controlled debit card commits the network to an **agent-cardholder regulatory posture** that Visa and Amex now have to mirror or cede share on. Watch for **Visa "AgentPay" or equivalent within 60 days**. (3) **Combined with the MoonPay-Exodus distribution partnership and ClawBank's "zero-human company" filing** (below), May 1, 2026 is the de facto **birthday of the U.S. AI-agent-economy commercial stack** — fiat rails, corporate identity, and crypto rails all closed within 24 hours of each other.
