### Anthropic Releases Claude Legal Plugin — Legal Tech Stocks Tumble
**Date:** February 2–3, 2026

Anthropic released its Claude Legal Plugin as part of the Cowork platform, prompting a significant selloff in legal technology stocks.

**What the Legal Plugin Does:**
- Automates contract review against configurable negotiation playbooks with green/yellow/red flags and redline suggestions
- `/triage-nda` command categorizes incoming NDAs for standard approval, counsel review, or full review
- `/vendor-check` checks vendor agreement status
- `/brief` generates contextual briefings
- `/respond` creates templated responses

**Open-Source Release:**
Anthropic released 11 open-source plugins spanning: Productivity, Enterprise Search, Sales, Finance, Data, Legal, Marketing, Customer Support, Product Management, and Biology Research.

**Market Impact:**
Shares of RELX (Lexis Nexis parent), Thomson Reuters (Westlaw), and LegalZoom fell sharply in premarket trading. Analysts note RELX and Thomson Reuters have proprietary data that may provide some insulation, while LegalZoom's business model may be more directly threatened.

**Availability:**
The plugin is open-source and free, but requires a paid Claude subscription (Professional, Team, or Enterprise) to use Cowork.

## Security

### Docker Patches Critical "DockerDash" AI Assistant Vulnerability
**Date:** February 2–3, 2026

Noma Labs disclosed details of "DockerDash," a critical vulnerability in Docker's Ask Gordon AI assistant that could have enabled code execution and data exfiltration through malicious container metadata.

**The Vulnerability:**
A malicious metadata label in a Docker image could compromise Docker environments through "Meta-Context Injection"—the AI fails to distinguish between benign metadata and malicious instructions.

**Attack Chain:**
1. Gordon AI reads and interprets malicious instruction from image metadata
2. Forwards it to the MCP (Model Context Protocol) Gateway
3. MCP Gateway executes through MCP tools

**Impact:**
- In environments with higher permissions: Remote Code Execution (RCE)
- In Docker Desktop (read-only permissions): High-impact data exfiltration—attackers could gather network topology and environment variables

**Mitigations (Docker Desktop 4.50.0, November 2025):**
- Ask Gordon no longer renders user-provided image URLs
- Explicit user confirmation now required before invoking any MCP tools

**Timeline:**
- September 17, 2025: Noma Labs discovered and reported
- October 13, 2025: Docker confirmed vulnerability
- November 6, 2025: Patch released in Docker Desktop 4.50.0
- February 2026: Full technical disclosure
