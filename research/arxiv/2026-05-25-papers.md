## Highlights (Top 5 Must-Read)

### 1. Practical Scaling Laws: Converting Compute into Performance in a Data-Constrained World
**Authors**: See arXiv:2605.09189
**arXiv**: https://arxiv.org/abs/2605.09189
**TL;DR**: Proposes a closed-form extension to Chinchilla that decomposes loss into undercapacity, undertraining, and overfitting terms, saturating between the irreducible loss and an uninformed baseline. Gives practitioners a way to convert a fixed compute budget into expected performance when high-quality data is scarce.
**Why it matters**: As frontier labs hit the data wall, a data-constrained successor to Chinchilla is directly actionable for allocating compute between model size and repeated tokens.

### 2. Continuous Latent Diffusion Language Model (Cola DLM)
**Authors**: See arXiv:2605.06548
**arXiv**: https://arxiv.org/abs/2605.06548
**TL;DR**: A hierarchical latent diffusion language model that first learns a stable text-to-latent mapping with a Text VAE, models a global semantic prior in continuous latent space with a block-causal DiT, then generates text via conditional decoding. Frames generation as hierarchical information decomposition rather than left-to-right token prediction.
**Why it matters**: A credible non-autoregressive architecture for text, pushing diffusion LMs toward parity with transformers on a different efficiency/quality frontier.

### 3. From Reasoning Chains to Verifiable Subproblems: Curriculum RL for Credit Assignment (SCRL)
**Authors**: See arXiv:2605.22074
**arXiv**: https://arxiv.org/abs/2605.22074
**TL;DR**: SCRL (Subproblem Curriculum Reinforcement Learning) derives verifiable subproblems from reference reasoning chains, enabling fine-grained credit assignment instead of a single sparse reward at the end of a long chain. The curriculum scales subproblem difficulty as training progresses.
**Why it matters**: Sparse end-of-chain rewards are a core bottleneck in RL for reasoning; decomposing chains into verifiable steps is a concrete fix to the credit-assignment problem.

### 4. Why Do Aligned LLMs Remain Jailbreakable: Refusal-Escape Directions
**Authors**: Yu Chen, Yuanhao Liu, Qi Cao
**arXiv**: https://arxiv.org/abs/2605.08878
**TL;DR**: Identifies "Refusal-Escape Directions" (RED) — local perturbation directions around a harmful input that flip the model from refusal to answering while preserving its harmful-semantics interpretation. Recasts jailbreaks as continuous behavior transitions along RED and analyzes the safety-utility trade-off.
**Why it matters**: A mechanistic, geometry-level account of why alignment remains brittle, moving beyond ad-hoc prompt attacks toward a structural understanding of refusal.

### 5. MemLens: Benchmarking Multimodal Long-Term Memory in Large Vision-Language Models
**Authors**: See arXiv:2605.14906
**arXiv**: https://arxiv.org/abs/2605.14906
**TL;DR**: A benchmark for memory in multimodal multi-session conversations: 789 questions across five memory abilities at four context lengths (32K-256K tokens). Evaluating 27 LVLMs and 7 memory-augmented agents shows strong short-context grounding that degrades sharply as conversations grow.
**Why it matters**: Long-term multimodal memory is a key gap for assistant-style agents, and MemLens quantifies how today's LVLMs fall short under realistic context growth.
