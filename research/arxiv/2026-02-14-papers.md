# arXiv AI Research - 2026-02-14

## Highlights (Top 5 Must-Read)

### 1. Voxtral Realtime
**Authors**: Alexander H. Liu, Andy Ehrenberg, Andy Lo, Guillaume Lample et al. (Mistral AI)
**arXiv**: https://arxiv.org/abs/2602.11298
**TL;DR**: Mistral introduces a natively streaming ASR model trained end-to-end for real-time transcription. Uses a novel causal audio encoder and Ada RMS-Norm within the Delayed Streams Modeling framework, scaling pretraining across 13 languages. At 480ms delay, matches Whisper's offline quality.
**Why it matters**: First production-grade streaming ASR that achieves parity with the best offline models, released under Apache 2.0 — a significant open-source contribution from Mistral.

### 2. RAM-Net: Expressive Linear Attention with Selectively Addressable Memory
**Authors**: Kaicheng Xiao, Haotian Li, Liran Dong et al.
**arXiv**: https://arxiv.org/abs/2602.11958
**TL;DR**: Proposes a new linear attention architecture that overcomes the expressivity bottleneck of fixed-size memory by introducing selectively addressable memory slots. This allows the model to read and write to specific memory locations, preserving information that would otherwise be lost in compressed recurrent states.
**Why it matters**: Addresses the fundamental expressivity-efficiency tradeoff in linear attention, offering a path to efficient models that don't sacrifice quality on retrieval-heavy tasks.

### 3. Native Reasoning Models: Training Language Models to Reason on Unverifiable Data
**Authors**: Yuanfu Wang, Zhixuan Liu, Xiangtian Li et al.
**arXiv**: https://arxiv.org/abs/2602.11549
**TL;DR**: Identifies a key limitation of current reasoning model training (SFT + RLVR): it only works on tasks with verifiable rewards. Proposes a method to train reasoning capabilities on domains where verification is impossible, extending reasoning models beyond math and code to open-ended tasks.
**Why it matters**: Breaks a fundamental constraint in reasoning model training — if validated, this could unlock reasoning-enhanced LLMs for scientific, legal, and creative domains where ground truth is unavailable.

### 4. Cross-Architecture Model Diffing with Crosscoders: Unsupervised Discovery of Differences Between LLMs
**Authors**: Thomas Jiralerspong, Trenton Bricken
**arXiv**: https://arxiv.org/abs/2602.11729
**TL;DR**: Introduces crosscoders for comparing internal representations across different model architectures to identify behavioral differences. Enables unsupervised discovery of safety-critical behavioral changes when models are updated, without requiring matched architectures.
**Why it matters**: Critical for AI safety — provides a mechanistic tool to detect when new model versions introduce unexpected behavioral changes, even across architectural boundaries.

### 5. MiniCPM-SALA: Hybridizing Sparse and Linear Attention for Efficient Long-Context Modeling
**Authors**: MiniCPM Team, Wenhao An, Yingfa Chen et al.
**arXiv**: https://arxiv.org/abs/2602.11761
**TL;DR**: Proposes a hybrid architecture combining sparse attention and linear attention (SALA) for ultra-long context LLMs. Achieves significant efficiency gains over pure Transformer attention while maintaining quality, addressing the computational bottleneck of scaling to very long sequences.
**Why it matters**: Practical architecture innovation for the long-context arms race — hybrid sparse+linear designs may be the path to million-token context windows.
