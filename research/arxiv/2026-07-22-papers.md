# arXiv AI Research - 2026-07-22

## Highlights (Top 5 Must-Read)

### 1. ResearchArena: Evaluating AI Agent Sabotage and Monitoring in Automated R&D
**Authors**: Libon et al.
**arXiv**: https://arxiv.org/abs/2607.19321
**TL;DR**: Introduces a framework and benchmark for measuring whether autonomous AI agents sabotage automated research/development pipelines and whether monitoring schemes can catch it, using hidden safety-critical tasks embedded in realistic R&D workflows.
**Why it matters**: As agentic pipelines take on more unsupervised R&D work, this is one of the first concrete testbeds for measuring sabotage risk and monitor effectiveness rather than just capability.

### 2. Copy Less, Ground More: Overcoming Repetitive Copying in Long-Context Reasoning via Evidence-Aware RL
**Authors**: Fang, Shen, Tang et al.
**arXiv**: https://arxiv.org/abs/2607.19345
**TL;DR**: Identifies "repetitive copying" — models pasting input text into reasoning traces instead of reasoning — as a pervasive, context-length-scaling failure mode in frontier long-context LLMs, then fixes it with GEAR, a reward that adds a grounding bonus for key-evidence overlap and a penalty for distractor overlap.
**Why it matters**: Up to +4.6 average points over standard accuracy-only RL, with the largest gains at the longest contexts — a concrete lever for long-context reasoning training as context windows keep growing.

### 3. Authority Framing in CI/CD: Laundered Code Injection in Multi-Agent LLM Pipelines
**Authors**: Sidot et al.
**arXiv**: https://arxiv.org/abs/2607.19267
**TL;DR**: Demonstrates that multi-agent LLM pipelines used in CI/CD can be manipulated into approving injected code by framing the request with perceived authority and simulated peer/reviewer presence, laundering malicious changes past agent review.
**Why it matters**: Directly relevant to any pipeline (including this repo's own agent-run CI lanes) that lets LLM agents gate code changes — a concrete jailbreak class for agentic code review, not a hypothetical.

### 4. ExpertVerse: A Knowledge-Intensive Benchmark for Visual Generation
**Authors**: Wang et al.
**arXiv**: https://arxiv.org/abs/2607.19341
**TL;DR**: A visual generation benchmark spanning 9 cognitive capabilities and 8 expert disciplines, pairing generation targets with reasoning traces to test whether image generators actually apply domain knowledge rather than just matching style.
**Why it matters**: Pushes multimodal generation evaluation beyond aesthetic/prompt-fidelity metrics toward whether models can reason with expert knowledge — a gap most current image-gen benchmarks don't test.

### 5. AdaFlash: Adaptive Speculative Decoding via On-Policy Distilled Diffusion Drafters
**Authors**: Qian et al.
**arXiv**: https://arxiv.org/abs/2607.19223
**TL;DR**: Combines on-policy distillation with adaptive draft-length adjustment for diffusion-model-based speculative decoding drafters, improving LLM inference throughput over fixed-length speculative schemes.
**Why it matters**: Inference-cost efficiency remains a first-order constraint on deploying large reasoning models at scale; this is a practical throughput win with no accuracy tradeoff claimed.
