## Highlights (Top 5 Must-Read)

### 1. Decoupled Mixture-of-Experts for Parametric Knowledge Injection
**Authors**: see arXiv listing (2606.14243)
**arXiv**: https://arxiv.org/abs/2606.14243
**TL;DR**: Proposes DMoE, a modular MoE that decouples both the experts and
the router from the frozen base model and attaches experts only to the
final-layer FFN. This preserves KV-cache reuse while enabling
parameter-level knowledge augmentation for efficient autoregressive
inference.
**Why it matters**: Shows you can inject new parametric knowledge into a
deployed LLM without retraining the backbone or breaking the KV-cache —
a practical path to updatable models.

### 2. Redesign Mixture-of-Experts Routers with Manifold Power Iteration
**Authors**: see arXiv listing (2606.12397)
**arXiv**: https://arxiv.org/abs/2606.12397
**TL;DR**: Reframes MoE routing as a spectral problem and replaces the
standard learned router with a Manifold Power Iteration (MPI) scheme that
selects experts per token from the geometry of the token manifold rather
than a softmax gate.
**Why it matters**: Targets the long-standing router-collapse / load-balance
failure modes of sparse MoE with a principled, less heuristic mechanism.

### 3. NRT-Bench: Multi-Turn Red-Teaming of Safety-Critical LLM Agents
**Authors**: see arXiv listing (2606.20408)
**arXiv**: https://arxiv.org/abs/2606.20408
**TL;DR**: Introduces a benchmark for multi-turn red-teaming of LLM agents
acting as operators of a safety-critical system, instantiated in a
simulated nuclear-power-plant control room with a five-role operator team,
each backed by a configurable LLM.
**Why it matters**: Moves agent-safety evaluation beyond single-turn
jailbreaks into realistic, multi-agent, high-stakes operational settings.

### 4. PP-OCRv6: Surpassing Billion-Scale VLMs on OCR at 34.5M Parameters
**Authors**: PaddleOCR team (2606.13108)
**arXiv**: https://arxiv.org/abs/2606.13108
**TL;DR**: A lightweight OCR system (1.5M→34.5M params) combining
architectural innovation with data-centric optimization that outperforms
billion-scale VLMs on OCR tasks, sidestepping VLM hallucination, imprecise
localization, and prohibitive compute.
**Why it matters**: A concrete reminder that specialized, tiny models still
beat general-purpose VLMs on narrow high-value tasks — strong efficiency story.

### 5. Stream3D-VLM: Online 3D Spatial Understanding with Incremental Geometry Priors
**Authors**: see arXiv listing (2606.06891)
**arXiv**: https://arxiv.org/abs/2606.06891
**TL;DR**: An online 3D vision-language model that does real-time spatial
understanding from streaming video via autoregressive next-token control and
a lightweight Visual-Spatial Feature Integration module that injects
temporally aligned geometry priors into the visual stream.
**Why it matters**: Pushes VLMs toward streaming, embodied 3D perception
rather than static single-image reasoning.
