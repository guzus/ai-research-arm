# arXiv AI Research - 2026-06-07

## Highlights (Top 5 Must-Read)

### 1. Goedel-Architect: Streamlining Formal Theorem Proving with Blueprint Generation and Refinement
**Authors**: Jui-Hui Chung, Ziyang Cai, Zihao Li, Hongzhou Lin, Mengdi Wang, Danqi Chen, Chi Jin, Sanjeev Arora, et al. (Princeton)
**arXiv**: https://arxiv.org/abs/2606.06468
**TL;DR**: An agentic Lean 4 framework that first generates a *blueprint* (a dependency graph of definitions and lemmas) and then closes each open lemma node in parallel, with failed lemmas driving global refinement — replacing the recursive lemma decomposition that loops on dead ends. Built on open-weight DeepSeek-V4-Flash, it hits 99.2% pass@1 on MiniF2F (100% with NL-proof seeding), 88.8% on PutnamBench, and solves 4/6 IMO 2025 and 3/6 USAMO 2026 problems.
**Why it matters**: State-of-the-art open-source automated theorem proving at up to 500x lower cost, suggesting blueprint-first planning beats recursive decomposition for long-horizon formal reasoning.

### 2. You Only Index Once: Cross-Layer Sparse Attention with Shared Routing
**Authors**: Yutao Sun, Yanqi Zhang, Li Dong, Jianyong Wang, Furu Wei (Microsoft Research)
**arXiv**: https://arxiv.org/abs/2606.06467
**TL;DR**: Cross-Layer Sparse Attention (CLSA) extends KV-sharing architectures (YOCO) by sharing not just the KV cache but the *routing index*: a single indexer computes token-level top-k selection once and reuses it across layers, keeping token-sparse accuracy while amortizing routing cost. Delivers up to 7.6x decoding speedup and 17.1x throughput improvement at 128K context.
**Why it matters**: Attacks all three long-context bottlenecks (prefill, KV storage, decode) jointly — a cleaner architectural answer to the efficiency-vs-quality trade-off in reasoning-heavy long-context inference.

### 3. Pretraining Recurrent Networks without Recurrence
**Authors**: Akarsh Kumar, Phillip Isola (MIT)
**arXiv**: https://arxiv.org/abs/2606.06479
**TL;DR**: Supervised Memory Training (SMT) trains nonlinear RNNs without backpropagation through time by reducing training to supervised learning on one-step memory-transition labels `(m_t, x_{t+1}) → m_{t+1}`, with labels obtained from a Transformer encoder trained on a predictive-state objective. This gives time-parallel training and a stable O(1)-length gradient path between any two tokens.
**Why it matters**: SMT outperforms BPTT when pretraining RNNs on language and pixel-sequence modeling, potentially unlocking parallel, scalable training of recurrent models that build temporal abstractions.

### 4. MLEvolve: A Self-Evolving Framework for Automated Machine Learning Algorithm Discovery
**Authors**: Shangheng Du, Xiangchao Yan, Jinxin Shi, Xin Li, Jie Zhou, Bo Zhang, Lei Bai, et al.
**arXiv**: https://arxiv.org/abs/2606.06473
**TL;DR**: An LLM-based self-evolving multi-agent system for end-to-end ML algorithm discovery. It replaces memoryless tree search with Progressive MCGS (cross-branch reference edges + an entropy-inspired exploration→exploitation schedule), adds a Retrospective Memory combining a cold-start knowledge base with dynamic global memory, and decouples strategic planning from code generation.
**Why it matters**: Achieves SOTA on MLE-Bench (medal rate, valid-submission rate) under a 12-hour budget — half the standard runtime — advancing autonomous ML engineering agents.

### 5. Latent Reasoning with Normalizing Flows (NF-CoT)
**Authors**: Guancheng Tu, Xiangjun Fu, Suhao Yu, Yao Tang, Haoqiang Kang, Lianhui Qin, Yizhe Zhang, Jiatao Gu
**arXiv**: https://arxiv.org/abs/2606.06447
**TL;DR**: NF-CoT performs intermediate reasoning in compact continuous "thoughts" modeled with a TARFlow-style normalizing flow inside the LLM backbone, while preserving the properties that make textual CoT work: native left-to-right generation, probabilistic sampling, KV-cache compatibility, and exact likelihoods. An NF head generates latent positions; the standard LM head generates text — within one causal stream.
**Why it matters**: A higher-bandwidth latent-reasoning alternative to discrete CoT that, unlike prior continuous-thought methods, keeps tractable likelihoods and autoregressive decoding intact.
