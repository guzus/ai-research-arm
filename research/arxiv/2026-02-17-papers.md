# arXiv AI Research - 2026-02-17

## Highlights (Top 5 Must-Read)

### 1. CogRouter: Step-Level Cognitive Depth Adaptation for LLM Agents
**Authors**: (arXiv:2602.12662)
**arXiv**: https://arxiv.org/abs/2602.12662
**TL;DR**: CogRouter is a framework grounded in ACT-R cognitive theory that trains agents to dynamically allocate cognitive depth across four hierarchical levels, from instinctive responses to strategic planning. Uses Cognition-aware Supervised Fine-tuning (CoSFT) followed by Cognition-aware Policy Optimization (CoPO) for step-level credit assignment.
**Why it matters**: With Qwen2.5-7B as base, achieves 82.3% success rate, outperforming GPT-4o (+40.3%) and OpenAI-o3 (+18.3%) while reducing token consumption by 62%.

### 2. MRPO: Expanding Reasoning Capacity via Manifold-Reshaping Policy Optimization
**Authors**: Dayu Wang, Jiaye Yang, Weikang Li, Jiahui Liang, Yang Li
**arXiv**: https://arxiv.org/abs/2602.02545
**TL;DR**: Challenges the hypothesis that RL merely aligns existing latent capabilities by demonstrating the latent reasoning space can be fundamentally expanded through geometric interventions. Proposes Spectral Orthogonal Exploration (SOE) to eject policy into the null space of the bias manifold, combined with Effective Rank regularization.
**Why it matters**: First convincing evidence that RL can genuinely expand LLM reasoning capacity beyond pre-trained boundaries, not just surface existing knowledge.

### 3. Elastic Memory: Compressive and Scalable Recurrent Memory
**Authors**: Yunchong Song, Jushi Kai, Liming Lu, Kaixi Qiu, Zhouhan Lin
**arXiv**: https://arxiv.org/abs/2602.11212
**TL;DR**: Introduces Elastic Memory, a novel memory architecture grounded in the HiPPO framework that treats historical sequences as samples from continuous signals, applying optimal online compression into a fixed-size memory state. Uses polynomial sampling for flexible retrieval from compressed representations.
**Why it matters**: Outperforms Memorizing Transformer with 16x less memory on 32k+ context benchmarks, offering a practical path to scalable long-context processing.

### 4. Accordion-Thinking: Self-Regulated Step Summaries for Efficient LLM Reasoning
**Authors**: (arXiv:2602.03249)
**arXiv**: https://arxiv.org/abs/2602.03249
**TL;DR**: End-to-end framework where LLMs learn to self-regulate reasoning granularity through dynamic summarization. The "Fold" inference mode periodically summarizes thought processes and discards former thoughts, reducing KV cache dependency. RL training progressively closes the accuracy gap between Fold and exhaustive Unfold modes.
**Why it matters**: Addresses the critical inference cost problem of long Chain-of-Thought reasoning by learning to compress reasoning context without accuracy loss.

### 5. CARE-RFT: Confidence-Anchored Reinforcement Finetuning for Reliable Reasoning
**Authors**: Shuozhe Li, Jincheng Cao, Bodun Hu, Aryan Mokhtari, Leqi Liu, Amy Zhang
**arXiv**: https://arxiv.org/abs/2602.00085
**TL;DR**: Replaces standard reverse KL regularization with skew reverse KL divergence in reinforcement finetuning. Provides bounded penalty on tokens the model is confident about (enabling reasoning exploration) while enforcing unbounded penalty elsewhere (preserving calibration).
**Why it matters**: Matches reasoning performance of unconstrained RFT while recovering the trustworthiness and calibration of the base model, solving the reasoning-vs-reliability tradeoff.
