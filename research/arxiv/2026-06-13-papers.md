# arXiv AI Research - 2026-06-13

## Highlights (Top 5 Must-Read)

### 1. MiniMax Sparse Attention
**Authors**: MiniMax
**arXiv**: [2606.13392](https://arxiv.org/abs/2606.13392)
**TL;DR**: MiniMax Sparse Attention (MSA) is a blockwise sparse attention built on Grouped Query Attention. A lightweight Index Branch scores key-value blocks and selects a Top-k subset per GQA group, then a Main Branch performs exact block-sparse attention over only the selected blocks — sidestepping the quadratic cost of softmax attention at million-token context.
**Why it matters**: A deployment-minded path to ultra-long context (agentic workflows, repo-scale code reasoning, persistent memory) that stays efficient across a broad range of GPUs.

### 2. End-to-End Context Compression at Scale (Latent Context Language Models)
**Authors**: (see arXiv listing)
**arXiv**: [2606.09659](https://arxiv.org/abs/2606.09659)
**TL;DR**: Introduces a family of latent-context compressors (e.g. a 0.6B encoder + 4B decoder, each continually pre-trained on 350B+ tokens) that push the Pareto frontier across general-task performance, compression speed, and peak memory.
**Why it matters**: Compressing context end-to-end at scale is a practical lever for cheaper, longer-context inference without a bespoke retrieval stack.

### 3. SABER: Benchmarking Operational Safety of LLM Coding Agents
**Authors**: (see arXiv listing)
**arXiv**: [2606.01317](https://arxiv.org/abs/2606.01317)
**TL;DR**: An environment-aware safety benchmark that drops models into realistic agent-style projects and scores safety from the *final environment state* after a sequence of actions. Even the best model exceeds a 54% harmful safety-violation rate.
**Why it matters**: Moves safety evaluation from single-prompt refusals to stateful, consequence-based outcomes — exposing how poorly current alignment holds in real agent workspaces.

### 4. ARM: An AutoRegressive Large Multimodal Model with Unified Discrete Representations
**Authors**: (see arXiv listing)
**arXiv**: [2606.11188](https://arxiv.org/abs/2606.11188)
**TL;DR**: A 7B discrete-representation autoregressive model that unifies image understanding, generation, and editing in a single architecture using unified discrete tokens for vision and language.
**Why it matters**: A clean single-stack design for any-to-any multimodal capability, avoiding separate understanding/generation pipelines.

### 5. From Player to Master: Test-Time Learning of LLM Agents via RL over Memory (MemoPilot)
**Authors**: (see arXiv listing)
**arXiv**: [2606.08656](https://arxiv.org/abs/2606.08656)
**TL;DR**: MemoPilot is a plug-in memory copilot that trains the *memory update process* of a frozen LLM, formulating memory updating as a multi-turn decision problem optimized end-to-end with multi-turn GRPO.
**Why it matters**: Decouples test-time learning from the base model — improving sequential-interaction performance without fine-tuning the LLM itself.
