# arXiv AI Research - 2026-07-10

## Highlights (Top 5 Must-Read)

### 1. Native Video-Action Pretraining for Generalizable Robot Control (LingBot-VA 2.0)
**Authors**: Qihang Zhang, Lin Li, Luyao Zhang, et al.
**arXiv**: https://arxiv.org/abs/2607.08639v1
**TL;DR**: A robot-control foundation model that combines semantic visual-action tokenization, causal video-action pretraining, and a sparse mixture-of-experts backbone to unify perception and control in one architecture. Reports strong few-shot generalization across manipulation tasks without task-specific fine-tuning.
**Why it matters**: A new architecture pattern for embodied foundation models, extending the "pretrain on video, act via tokens" recipe with MoE scaling.

### 2. Super Weights in LLMs and the Failure of Selective Training
**Authors**: Shreyas Subramanian, Adewale Akinfaderin, Akarsha Sehgag
**arXiv**: https://arxiv.org/abs/2607.08733v1
**TL;DR**: Shows that training only the small set of "Super Weights" identified as most important collapses accuracy to random-chance levels, while vanilla LoRA using just 0.16% of parameters succeeds — parameter importance does not imply trainability in isolation.
**Why it matters**: A sharp counterexample to "importance = trainability" assumptions behind selective fine-tuning and pruning heuristics used across the efficiency literature.

### 3. Switch-Reasoner: Learn When to Think in Multitask Mixtures via Reinforcement Learning
**Authors**: Yiyang Fang, Pei Fu, Jinjie Li, et al.
**arXiv**: https://arxiv.org/abs/2607.08572v1
**TL;DR**: A GRPO-based training framework that lets multimodal LLMs adaptively choose between explicit chain-of-thought reasoning and direct answering per-task, cutting inference cost while holding accuracy.
**Why it matters**: Concrete progress on adaptive test-time compute — one of the highest-leverage open problems for reasoning-model efficiency.

### 4. Remember When It Matters: Proactive Memory Agent for Long-Horizon Agents
**Authors**: Yifan Wu, Lizhu Zhang, Yuhang Zhou, et al.
**arXiv**: https://arxiv.org/abs/2607.08716v1
**TL;DR**: Introduces a memory module that runs alongside an action agent, maintaining structured memory and selectively injecting reminders rather than passively retrieving context — improves pass@1 and mitigates "behavioral state decay" over long task horizons.
**Why it matters**: Long-horizon agentic memory is a core bottleneck for autonomous agents, and proactive (vs. passive-retrieval) memory is a meaningfully different design point.

### 5. UniClawBench: A Universal Benchmark for Proactive Agents on Real-World Tasks
**Authors**: Zhekai Chen, Chengqi Duan, Kaiyue Sun, et al.
**arXiv**: https://arxiv.org/abs/2607.08768v1
**TL;DR**: A capability-driven benchmark for proactive agents spanning 400 bilingual tasks in live Docker containers, with fine-grained completion checkpoints and closed-loop multi-turn human-feedback simulation.
**Why it matters**: Moves agent evaluation beyond static single-turn tasks toward dynamic, proactive, real-world-shaped settings — the eval surface most agent benchmarks still lack.
