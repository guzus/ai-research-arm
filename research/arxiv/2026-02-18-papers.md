# arXiv AI Research - 2026-02-18

## Highlights (Top 5 Must-Read)

### 1. Selective Synchronization Attention
**Authors**: (arXiv:2602.14445)
**arXiv**: https://arxiv.org/abs/2602.14445
**TL;DR**: Replaces standard dot-product self-attention with a closed-form operator derived from the Kuramoto model of coupled oscillators. Each token is an oscillator with learnable frequency and phase; synchronization strength serves as attention weight. Instantiated in the Oscillatory Synchronization Network (OSN), a drop-in Transformer block replacement.
**Why it matters**: Introduces natural sparsity from phase-locking thresholds, eliminates separate positional encodings, and avoids iterative ODE integration — a fundamentally new attention paradigm bridging neuroscience and deep learning.

### 2. Reinforcement Inference: Self-Correcting LLM Reasoning via Uncertainty
**Authors**: (arXiv:2602.08520)
**arXiv**: https://arxiv.org/abs/2602.08520
**TL;DR**: An entropy-aware inference-time strategy that detects when an LLM is uncertain about its first answer and selectively triggers a second, more deliberate reasoning attempt — no retraining needed. On MMLU-Pro (12K questions), improves DeepSeek-v3.2 accuracy from 60.72% to 84.03% with only 61% additional inference calls.
**Why it matters**: Demonstrates that many LLM errors stem from premature commitment under ambiguity, not missing knowledge — a practical, training-free approach to unlock latent reasoning capability.

### 3. CARE-RFT: Confidence-Anchored Reinforcement Finetuning
**Authors**: Shuozhe Li, Jincheng Cao, Bodun Hu, Aryan Mokhtari, Leqi Liu, Amy Zhang
**arXiv**: https://arxiv.org/abs/2602.00085
**TL;DR**: Replaces standard reverse KL regularization in reinforcement finetuning with skew reverse KL divergence. The penalty is bounded for confident, consistently rewarded explorations (enabling reasoning) but unbounded elsewhere (preserving calibration). Matches unconstrained RFT reasoning gains while recovering base model trustworthiness.
**Why it matters**: Solves the fundamental tension between reasoning performance and model calibration in reinforcement finetuning — critical for deploying reasoning models safely.

### 4. Elastic Memory: Compressive and Scalable Recurrent Memory
**Authors**: Yunchong Song et al.
**arXiv**: https://arxiv.org/abs/2602.11212
**TL;DR**: A novel recurrent memory architecture grounded in HiPPO theory that treats historical sequences as continuous signals and applies optimal online compression into fixed-size memory states. Outperforms Memorizing Transformer with 16x less memory and beats Melodi at all memory sizes on 32K+ long-document tasks.
**Why it matters**: Bridges the theoretical elegance of HiPPO with practical scalability, offering a principled alternative to attention-based long-context approaches.

### 5. Information Fidelity in MCP Agents: A Martingale Analysis
**Authors**: (arXiv:2602.13320)
**arXiv**: https://arxiv.org/abs/2602.13320
**TL;DR**: First theoretical framework analyzing error accumulation in Model Context Protocol agents. Proves cumulative distortion grows linearly with high-probability deviations bounded by O(√T). Validated across Qwen2-7B, Llama-3-8B, and Mistral-7B. Derives optimal re-grounding intervals and monitoring strategies.
**Why it matters**: As MCP adoption accelerates, this work provides the mathematical foundation for safe deployment of tool-using agents in safety-critical domains.
