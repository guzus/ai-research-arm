## Breaking News

### Claude Goes GA in Microsoft Foundry on Azure — Anthropic's First-Ever Deployment on NVIDIA GPUs

**Date:** June 29, 2026 | **Sources:** [Claude blog — Claude in Microsoft Foundry is now generally available](https://claude.com/blog/claude-in-microsoft-foundry), [Microsoft Azure Blog — Introducing Anthropic's Claude models in Microsoft Foundry](https://azure.microsoft.com/en-us/blog/introducing-anthropics-claude-models-in-microsoft-foundry-bringing-frontier-intelligence-to-azure/), [Anthropic — Microsoft, NVIDIA and Anthropic announce strategic partnerships](https://www.anthropic.com/news/microsoft-nvidia-anthropic-announce-strategic-partnerships), [Windows News — Claude AI on Azure Foundry powered by NVIDIA GB300 Blackwell Ultra](https://windowsnews.ai/article/anthropic-launches-claude-ai-on-microsoft-azure-foundry-powered-by-nvidia-gb300-blackwell-ultra-gpus.432118), [@claudeai](https://x.com/claudeai/status/2071653958905467027)

- **What's net-new:** Anthropic and Microsoft announced **Claude Opus 4.8 and
  Claude Haiku 4.5 are now generally available in Microsoft Foundry**, hosted
  natively on **Azure** with Azure authentication, billing, and commitment
  retirement (prompt caching + extended thinking supported). The models run on
  **NVIDIA GB300 NVL72 (Blackwell Ultra)** systems with Quantum-X800 InfiniBand —
  reportedly **the first time Anthropic models are deployed on NVIDIA silicon**
  (Anthropic's stack has historically leaned on AWS Trainium and Google TPUs).
- **Why it matters:** Claude is now native across **AWS Bedrock, Google Vertex,
  and Azure Foundry** — making Microsoft the only cloud offering both OpenAI and
  Anthropic frontier models on the same platform. Anthropic is converting model
  scarcity into omni-cloud reach rather than committing to a single cloud patron.
- **Skeptic's note:** "GA in Foundry" is primarily a procurement/billing/governance
  integration — Claude was already reachable on Azure indirectly. The genuinely
  new facts are native Azure auth/billing and the first NVIDIA-GPU lane. The
  "first-ever on NVIDIA" superlative was initially single-sourced (@firstadopter,
  "I am told") but is consistent with NVIDIA's and Azure's own GB300 framing.

### The Frontier Gate Reverses — US Clears Mythos 5 for ~100 Orgs; Fable 5 Reported Returning "This Week"

**Date:** June 26–29, 2026 | **Sources:** [Fortune — Anthropic's Mythos 5 cleared by U.S. for wider use](https://fortune.com/2026/06/27/anthropic-mythos-5-ai-model-us-commerce-department-clearance-fable/), [CNBC — Trump admin allows Anthropic to release Mythos to some companies, agencies](https://www.cnbc.com/2026/06/26/us-government-anthropic-claude-mythos5-ai.html), [9to5Mac — Cleared to release Mythos 5 to over 100 US institutions](https://9to5mac.com/2026/06/26/anthropic-cleared-to-release-claude-mythos-5-to-over-100-us-institutions/), [The Hill — Federal government permits release of Mythos to select companies](https://thehill.com/policy/technology/5943549-anthropic-mythos-5-access/), [Let's Data Science — Anthropic restores Fable 5 after US ban](https://letsdatascience.com/news/anthropic-restores-fable-5-after-us-ban-9d5e2250)

- **What's net-new (vs. prior reports, which covered only the June 12
  *suspension*):** Commerce Secretary **Howard Lutnick's June 26 letter** (first
  reported by Semafor) cleared **Mythos 5** — Anthropic's stronger frontier
  sibling — for **~100 "trusted partner" US companies and federal agencies**,
  with Lutnick noting he can amend the approved list "at any time." **Axios**
  (June 27) reported **Fable 5**, offline 15 days, was on track to return within
  days as negotiations progressed.
- **Conflicting timing:** Lutnick's letter itself did **not** lift Fable 5's
  restriction (Pentagon/NSA sign-off reportedly still outstanding as of June 28),
  even as the Axios scoop and an Anthropic exec said Fable 5's return is imminent.
  Treat Fable 5 re-activation as **reported-imminent, not confirmed live**;
  the Mythos 5 clearance is the firmer fact.
- **Why it matters:** This is the direct counter-movement to last week's
  "Washington gates the frontier." The mechanism is now visibly **two-way** —
  gate, then selectively un-gate to vetted users — a *managed-access licensing
  regime* rather than a permanent ban, applied to the same weights that are
  simultaneously going omni-cloud (above).
