**Date:** Reported June 18–20, 2026 | **Sources:** [TestingCatalog — OpenAI prepares GPT-5.6 models for the upcoming release](https://www.testingcatalog.com/openai-prepares-gpt-5-6-models-for-the-upcoming-release/), [TechTimes — GPT-5.6: OpenAI Chief Scientist Calls It a Meaningful Leap](https://www.techtimes.com/articles/318492/20260616/gpt-56-openai-chief-scientist-calls-it-meaningful-leap-june-launch-nears.htm), [explainx.ai — GPT-5.6 Release Date, Features & Benchmarks (2026)](https://www.explainx.ai/blog/gpt-5-6-release-date-features-benchmarks-2026)

- **What's net-new (vs. the prior window's "Polymarket 83% on June 22–28"):** Reporting now points to OpenAI **widening the GPT-5 line next week**, with the **GPT-5.6 family poised for a Tuesday arrival** (~**June 23**) — and the standard model, **Mini**, and **Pro** variants reportedly **landing together** rather than trickling out. This is a concrete, near-term launch report sharpening the prior probabilistic window.
- **Leaked specs (unverified):** a context window pushed toward **1.5 million tokens** (up from the ~1M GPT-5.5 shipped in April), sharper long-horizon coding, and quicker **Codex** response times. Developers reported probing an unreleased `gpt-5.6` model inside Codex via ChatGPT Pro OAuth; routing-log hints first appeared April 28.
- **Critical caveat:** **OpenAI has not confirmed GPT-5.6 as a product.** No system card, no API model string, no official date. All specs (1.5M context, codenames, benchmarks, pricing) remain **rumor/leak**, and the "Tuesday next week" timing is press reporting, not an OpenAI announcement.
- **Why it matters:** If accurate, this would be the next frontier launch in a firehose June — and would land squarely into the ~June 30 cliff alongside the still-unshipped Gemini 3.5 Pro GA and Grok 5.
