- **Benchmark deltas vs Opus 4.7 (Anthropic's published metrics):**
  - **Agentic coding:** **64.3% → 69.2%** (+4.9pp)
  - **Multidisciplinary reasoning with tools:** **54.7% → 57.9%** (+3.2pp)

- **Dynamic Workflows (research preview, in Claude Code):** A new orchestration primitive where **Claude writes a JavaScript script that orchestrates subagents at scale**. A runtime then executes the script in the background. **A single run can spawn up to 1,000 parallel agents.** Anthropic's framing: Claude Code on Opus 4.8 with Dynamic Workflows can now carry out **codebase-scale migrations across hundreds of thousands of lines of code from kickoff to merge**, with the **existing test suite as the merge bar**.

The benchmark deltas are the **proof-of-flow-state** Anthropic needs to justify the **~20.5x forward revenue multiple** to its new equity holders. (2) **Dynamic Workflows is the direct architectural answer to OpenAI's Codex Goal Mode (GA May 27, carry from the May 28 report) and Cognition Devin** — both of which pitched "persistent agentic loops" as the differentiator. By moving to **JavaScript-orchestrated 1,000-subagent parallelism**, Anthropic is jumping a layer: the comparison is no longer "loop vs one-shot" but **"orchestrated swarm vs single-agent loop."** The 1,000-agent cap also creates an explicit **cost-bounding contract** — useful for enterprise procurement, which has been the bottleneck on Claude Code adoption in regulated industries. (3) **The pricing-held-flat + Fast-Mode-3x-cheaper is a defensive posture against the open-weight pressure (DeepSeek V4 Pro, Qwen 3.7 Max, Llama 4)** — Anthropic is willing to compress Opus margins to keep Opus 4.8 at price parity with the Tier 2 reasoning models from China, while preserving headroom for **Mythos-class** as the true premium SKU. Simon Willison's read ("a modest but tangible improvement") is the right frame: Opus 4.8 is a **flow-state release**, not a frontier-architecture release, and the **Mythos line is what Anthropic is saving the architecture story for.**
