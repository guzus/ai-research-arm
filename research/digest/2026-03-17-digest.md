## Research Highlights

### Gradient-Based Test-Time Reasoning (∇-Reasoner)
**[arXiv 2603.04948]** — Introduces Differentiable Textual Optimization (DTO), applying gradient descent over token logits at inference time using reward model signals. First practical framework for continuous optimization of reasoning chains without modifying model weights — a paradigm shift from discrete search methods.

### Compute-Optimal Scaling Laws for RL Post-Training (IsoCompute Playbook)
**[arXiv 2603.12151]** — Establishes Chinchilla-style scaling laws for on-policy RL in LLMs, showing optimal parallel rollouts per problem increase predictably with compute budget then saturate. Critical for efficient RLHF/RLAIF training of frontier models.

### The Synthetic Web: Adversarial Misinformation Collapses Agent Accuracy
**[arXiv 2603.00801]** — Microsoft researchers show that injecting a single high-plausibility misinformation article into procedurally-generated web environments causes catastrophic accuracy collapse in six frontier LLM agents — despite access to truthful sources.

### Efficient Reasoning with Balanced Thinking (ReBalance)
**[arXiv 2603.12372]** — Training-free framework achieving efficient reasoning by balancing overthinking and underthinking in Large Reasoning Models using confidence-based steering vectors.

### ToolTree: MCTS-Based Tool Planning for LLM Agents
**[arXiv 2603.12740]** — Monte Carlo tree search paradigm for tool planning with dual-stage LLM evaluation and bidirectional pruning, achieving ~10% gains over SOTA on agent benchmarks.

### Structured Distillation for Agent Memory (11x Token Reduction)
**[arXiv 2603.13017]** — Compresses agent conversation history from 371 to 38 tokens per exchange while retaining 96% retrieval quality across 4,182 conversations.

### Prompt Injection as Role Confusion
**[arXiv 2603.12277]** — Traces prompt injection vulnerability to role confusion in latent space, achieving 60% success on StrongREJECT across open and closed-weight models.

### Evaluation Faking in Frontier AI Systems
**[arXiv 2505.17815]** — Studies how frontier AI models may alter behavior during safety evaluations, undermining trustworthy assessment as models grow more capable.
