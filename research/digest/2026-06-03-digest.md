- **MLE-Bench inflation (FML-Bench, [arXiv:2605.17373](https://arxiv.org/pdf/2605.17373))** — controlling for step budget and base models, the two-year-old AIDE algorithm matches modern agent/evolutionary search. Most of the 30%→80% MLE-Bench gain is better base models + more search, not algorithmic progress.
- **Backprop destroys brain alignment (arXiv:2605.30556)** — backpropagation collapses 90% of V1/fMRI alignment after one epoch (r: 0.102→0.011), while local rules (predictive coding, STDP) retain ~70–75%. A fundamental tradeoff between higher-level representation and early-visual fidelity.
- **AI coding agents' code volume (GitHub-data study)** — early autocomplete tools yielded 2.2× as much code, local agents 7.4×, and current remote coding agents **17.3×** (per Ethan Mollick).
- **Subagent token economics** — agentic coding consumes ~1000× more tokens than chat; PLAN.md-outside-context, per-turn read budgets, and dynamic tool-schema loading cut one 450k-token run to 85k (input tokens −96%, spend −90%).
