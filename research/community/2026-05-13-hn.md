### Notable Discussions

- **Needle — 26M tool-calling model** (579 pts / 167 comments): Cactus Compute open-sourced a 26M-parameter model distilled from Gemini's tool-calling behavior, aimed at on-device agentic workflows. Discussion centers on whether targeted distillation is the realistic path for small-model agents. [HN](https://news.ycombinator.com/item?id=48111896)
- **LLMs corrupt your documents when you delegate** (478 pts / 201 comments): arXiv paper measuring fidelity loss when LLMs edit documents in place; community debating safe pipeline patterns (re-read vs. mutate). [HN](https://news.ycombinator.com/item?id=48073246)
- **Maryland $2B grid for out-of-state AI** (318 pts / 201 comments): State filing with FERC challenging cost allocation to ratepayers for transmission feeding data centers serving neighboring states — flagship instance of the AI-power-cost fight. [HN](https://news.ycombinator.com/item?id=48088151)
- **Reimagining the mouse pointer for the AI era** (DeepMind, 239 pts / 206 comments): Proposal for a pointer that signals intent + context to an agent layer rather than raw coordinates; mostly skeptical reception. [HN](https://news.ycombinator.com/item?id=48111581)
- **Training an LLM in Swift, part 1** (258 pts): Detailed walk-through pushing Swift matmul from Gflop/s to Tflop/s on Apple silicon, with concrete numbers. [HN](https://news.ycombinator.com/item?id=48085685)
- **Statewright — state machines for agents** (108 pts / 49 comments): Open-source tool framing agent workflows as explicit state machines; compared favorably to natural-language-only agent designs (echoed by the "clipboard pattern" thread). [HN](https://news.ycombinator.com/item?id=48108778)
- **OpenGravity — BYOK Antigravity clone** (102 pts): Zero-install JS port of Antigravity, BYOK; getting traction as a minimal alternative for tinkerers. [HN](https://news.ycombinator.com/item?id=48100192)
- **Can LLMs model real-world systems in TLA+?** (122 pts / 32 comments): SIGOPS analysis of LLM-generated formal specs — useful for sketches, fails on invariants/fairness. [HN](https://news.ycombinator.com/item?id=48065254)
- **Hopper — agent for mainframes/COBOL** (83 pts / 43 comments): Hypercubic agent driving legacy COBOL/mainframe environments. [HN](https://news.ycombinator.com/item?id=48111143)
- **GLiGuard — small LLM guardrail** (35 pts): Pioneer.ai (GLiNER team) shipped a small-model safety moderator claimed at 16× faster than incumbents. [HN](https://news.ycombinator.com/item?id=48112544)
- **AMÁLIA — European Portuguese LLM** (140 pts / 80 comments): Sovereign-language model effort, discussion on the trade-off between training from scratch vs. fine-tuning multilingual base models. [HN](https://news.ycombinator.com/item?id=48065429)
- **FairyFuse — multiplication-free LLM inference on CPU** (21 pts): Fused ternary kernel approach targeting CPU-only deployments. [HN](https://news.ycombinator.com/item?id=48111527)
- **Voker — agent analytics** (55 pts / 20 comments): YC S24 launch focused on observability for production agents. [HN](https://news.ycombinator.com/item?id=48109962)
