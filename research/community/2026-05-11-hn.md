- **Local AI needs to be the norm** (1115 pts): Manifesto arguing local/on-device inference should displace cloud-only AI; debate over privacy, latency, hardware costs, and the role of small open models. Now the top HN story by a wide margin. https://news.ycombinator.com/item?id=48085821

- **PS3 Emulator Devs ask to stop AI PRs** (163 pts): RPCS3 maintainers push back on a flood of low-quality LLM-generated pull requests; thread explores the maintenance tax of "vibe-coded" contributions in OSS. https://news.ycombinator.com/item?id=48089263
- **AI coding agents must reduce maintenance costs** (156 pts): James Shore argues evaluating coding agents on lines-shipped is backwards — the real metric is long-term maintenance burden. https://news.ycombinator.com/item?id=48089289
- **Gemini API File Search now multimodal** (147 pts): Google's managed RAG endpoint now ingests images and video alongside text; commenters compare against Anthropic's file API and OpenAI's retrieval. https://news.ycombinator.com/item?id=48080702
- **Chrome's AI features hogging 4GB** (108 pts): Gemini Nano on-device weights silently inflate Chrome's footprint by ~4GB; thread debates opt-out, disk pressure on small SSDs, and the future of bundled local models. https://news.ycombinator.com/item?id=48084710
- **Claude as a user-space IP stack** (75 pts): Adam Dunkels (uIP/lwIP author) measures ping latency when an LLM hand-rolls the IP stack — a delightfully absurd benchmark of LLM-as-runtime. https://news.ycombinator.com/item?id=48089049
